# SmartHome
Este proyecto consiste en una Landing Page responsiva que simula un panel de control inteligente (Smart Home). La interfaz ha sido desarrollada bajo la filosofía Mobile-First, garantizando una experiencia óptima desde dispositivos móviles hasta pantallas de escritorio profesionales.
